About the Project

Project Purpose

The AI Image Authenticity Detector is designed to tackle the growing challenge of synthetic media and deepfakes. By leveraging advanced deep learning techniques, this tool provides a fast, reliable way to distinguish between real photographs and AI-generated images, ensuring trust and authenticity in digital media.

The Dataset

The CIFAKE dataset is used in this project to train and evaluate the AI Image Detection System for classifying images as real or fake. It consists of a balanced collection of real images and AI-generated images, allowing the model to learn meaningful differences between authentic and synthetic visual content. The dataset helps the deep learning model identify patterns such as texture inconsistencies and unnatural features commonly found in generated images. By training on CIFAKE, the model achieves better accuracy and generalization, making it effective for real-world image authenticity detection tasks.

Model Architecture

The architecture of the model used in this project is based on a combination of Convolutional Neural Networks (CNN) and the ResNet (Residual Network) approach. CNN is used as the fundamental building block for extracting important features from images, such as edges, textures, and patterns. It consists of multiple layers including convolutional layers, activation functions, pooling layers, and fully connected layers, which work together to progressively learn complex representations of the input images.

Technologies Used

  • Flask - Backend API & Server
  • HTML5 / CSS3 - User Interface
  • Vanilla JS - Dynamic Interactions
